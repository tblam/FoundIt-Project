1. IPL EC with instructions

2. Check for valid IP address

3. Create a node for the EC

4. Bind the @db2ubind.lst as db2clps.bnd file is in the list.

From DB2 command environment issue the following commands:
db2 terminate
db2 connect to <dbname> user <uid> using <pwd>

<dbname> is the name of your database
<uid> is your user id with bind privilege
<pwd> is your users password
cd {DB2InstallDir}/sqllib/bnd
db2 bind @db2cli.lst blocking all grant public sqlerror continue
db2 bind @db2ubind.lst blocking all grant public sqlerror continue
